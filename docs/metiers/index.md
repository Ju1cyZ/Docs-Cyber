# Métiers de la cybersécurité

Cette section accueillera des fiches détaillées pour les différents métiers de la cybersécurité (analyste SOC, pentester, ingénieur sécurité, répondant aux incidents, etc.).

Pour l'instant il n'y a pas de fiches listées — fournissez-moi la liste des métiers que vous voulez ajouter et je créerai une page dédiée pour chaque métier avec :

- description du rôle
- compétences requises
- outils courants
- parcours de formation et certifications recommandées

Structure prévue : chaque métier aura sa propre page sous `docs/metiers/<metier>.md` et la navigation sera mise à jour automatiquement.
